Prompt Co-Pilot Icon Generator

How to Create Icons:

  1. Use your browser's screenshot tool or extension to capture each icon below
  2. Crop precisely to the icon boundaries (16x16, 48x48, 128x128 pixels)
  3. Save the files as:
    • icon16.png
    • icon48.png
    • icon128.png
  4. Place all three files in the assets/icons/ folder

Alternative: Use an online icon generator like favicon-generator.org or design custom icons in Figma/Canva.
